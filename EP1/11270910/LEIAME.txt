Para Executar o Programa é necessário:

1) Abrir o terminal de execução;

2) Acessar a pasta src, onde se encontram os códigos-fonte pelo terminal, para isso você tem duas opções: 
   a) viajar da pasta atual até a pasta src pelo terminal com o comando cd (Linux)
   b) definir um classpath, com o comando java -classpath seguido do caminho da pasta atual até a pasta src;

3) Compilar Player.java, com o comando javac Player.java;

4) Compilar Score.java, com o comando javac Score.java;

5) Compilar Ball.java, com o comando javac Ball.java;

6) Rodar o Programa com java Pong;
